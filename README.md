# 🚀 NASA SpaceScope Dashboard

A real-time interactive data dashboard built using Python and Streamlit that fetches live space data from NASA APIs.

## 🌐 Live Demo

👉 Click here to view the app:
🔗 https://vishalg9670-nasa-spacescope-app-slrj7e.streamlit.app/

## 🌌 Features

* 📷 Astronomy Picture of the Day (APOD)
* ☄️ Asteroid tracking and monitoring
* 📊 Interactive visualizations using Plotly
* 🔍 User-friendly interface with dynamic data
* 🌐 Real-time API integration

## 🛠️ Tech Stack

* Python
* Streamlit
* REST API
* Plotly
* Pandas

## 🚀 How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 💡 Use Case

This project demonstrates how to build and deploy a real-time data analytics dashboard using APIs and Python.
